# Incident Response Dashboard (Angular)

This project is an Angular-based scaffold for incident and threat management workflows.

## Development server

```bash
npm install
npm start
```

Navigate to `http://localhost:4200/`.

## Build

```bash
npm run build
```

## Troubleshooting

If you see `ng: not found` when running `npm start` or `npm run build`, it means Angular CLI dependencies are not installed locally yet.

From the project root (folder containing `package.json`), run:

```bash
npm install
```

Then retry:

```bash
npm start
npm run build
```

## Included sections

- Dashboard
- Incidents / Incident Response
- Threats and Predictions
- Remediation and Playbook Creator
- Activity Logs, Billing, Systems, Settings, AI Ingestion
